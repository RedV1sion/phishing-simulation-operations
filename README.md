
# Enterprise Phishing Campaign Operations

## Overview

This repository contains simulated enterprise phishing campaign operations conducted using KnowBe4 within a Microsoft 365 enterprise environment.

The phishing operations program was designed to evaluate user susceptibility to phishing attacks, improve organizational security awareness, and reduce phishing-related risks through recurring simulations and remediation-focused training.

# Campaign Objectives

The phishing operations program focused on:

- Measuring phishing-prone percentage (PPP)
- Identifying high-risk users
- Simulating realistic phishing attack scenarios
- Improving phishing reporting behavior
- Delivering remediation training
- Tracking recurring phishing failures
- Reducing organizational phishing risk

| Campaign Type | Description |
