# Purpose

The purpose of this report is to identify high-risk user behaviors observed during phishing simulation campaigns and evaluate organizational phishing susceptibility trends.

This analysis supports:

- Security awareness improvement
- Risk-based remediation
- Phishing-prone user identification
- Department risk analysis
- Enterprise risk reduction initiatives

---

# Risk Scoring Methodology

User risk scores were calculated using:

- Phishing link click activity
- Credential submission attempts
- Repeat phishing failures
- Suspicious email reporting behavior
- Awareness training completion
- Campaign participation trends

# Remote Workforce Risk Trends

Remote and hybrid users demonstrated elevated phishing susceptibility due to:

- Heavy reliance on email communication
- Increased trust in Microsoft notifications
- Use of unmanaged personal devices
- Reduced in-person IT verification opportunities
- Increased MFA-related phishing exposure

---

# High-Risk Campaign Themes

The following phishing themes generated the highest interaction rates among high-risk users:

| Campaign Theme | Risk Impact |
|---|---|
| Security Breach Alerts | Very High |
| MFA Reset Requests | High |
| HR Policy Notifications | High |
| Invoice Payment Requests | Medium |
| Password Expiration Alerts | Medium |

---

# Credential Harvesting Trends

Credential harvesting simulations revealed:

- Users were more likely to submit credentials during:
  - Security alert campaigns
  - IT impersonation campaigns
  - MFA reset phishing
  - Microsoft-branded phishing scenarios

MFA deployment significantly reduced successful credential submission impact.

- Limited sender verification practices
- Increased urgency-driven decision making
