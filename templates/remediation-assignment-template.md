# Remediation Assignment Templates

## Enterprise Security Awareness Program
## Phishing Remediation Assignment Communications

---

# Overview

This document contains remediation assignment templates used within an enterprise phishing awareness and security operations program.

These templates are used to notify users who failed phishing simulations and assign targeted awareness training based on phishing-related risk behaviors.

The templates support:

- User remediation workflows
- Phishing awareness improvement
- Repeat offender management
- Security awareness enforcement
- Enterprise risk reduction

---

# Template 1 — Credential Harvesting Failure

## Subject

```text
Required Security Awareness Training Assignment
```

---

## Email Template

```text
Hello,

As part of our ongoing security awareness program, you recently interacted with a simulated phishing email involving credential harvesting techniques.

To help improve phishing awareness and reduce organizational security risk, additional awareness training has been assigned to your account.

Assigned Training Topics:
- Credential Harvesting Awareness
- Microsoft 365 Phishing Protection
- MFA Security Awareness
- Suspicious Link Identification

Please complete the assigned training within 7 days.

If you have questions or require assistance, please contact the Security Operations Team.

Thank you,
Security Operations Team
```

# Remediation Assignment Workflow

```text
User Fails Phishing Simulation
            ↓
Failure Type Categorized
            ↓
Risk Score Updated
            ↓
Targeted Training Assigned
            ↓
User Completes Awareness Training
            ↓
Follow-Up Simulation Scheduled
            ↓
User Risk Reassessed
```
