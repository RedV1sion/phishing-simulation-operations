# Template 1 — General Phishing Awareness Reminder

## Subject

```text
Security Reminder: Stay Alert for Phishing Emails
```

---

## Email Template

```text
Hello Team,

Cybercriminals continue to target organizations using phishing emails designed to steal credentials, financial information, and sensitive organizational data.

Please remain cautious when reviewing emails involving:

- Password reset requests
- MFA verification prompts
- Invoice payment requests
- Suspicious attachments
- Urgent security alerts
- Requests for sensitive information

Before clicking links or opening attachments:

- Verify the sender
- Check URLs carefully
- Report suspicious emails immediately
- Contact IT if unsure

If you receive a suspicious email, please use the Outlook phishing reporting button or contact the Security Team.

Thank you for helping protect our organization.

Security Operations Team
```

# Template 2 — Credential Harvesting Awareness

## Subject

```text
Important Security Reminder: Protect Your Microsoft 365 Credentials
```

---

## Email Template

```text
Hello Team,

Attackers frequently impersonate Microsoft 365 login pages to steal employee credentials.

Please remember:

- IT will never ask for your password by email
- Verify login pages before entering credentials
- Be cautious of urgent password expiration messages
- Report suspicious login requests immediately

Common phishing themes include:

- Password expiration alerts
- MFA reset notifications
- Suspicious login warnings
- Security breach notifications

If you believe you entered credentials into a suspicious website, contact the Security Team immediately.

Security Operations Team
```

# Template 6 — Phishing Reporting Awareness

## Subject

```text
Security Reminder: How to Report Suspicious Emails
```

---

## Email Template

```text
Hello Team,

Prompt reporting of suspicious emails helps protect the organization from phishing attacks and credential compromise.

Please report suspicious emails if you notice:

- Unusual sender addresses
- Urgent requests
- Suspicious links
- Unexpected attachments
- Requests for sensitive information

To report suspicious emails:

1. Use the Outlook phishing reporting button
2. Forward the email to the Security Team
3. Contact IT if immediate assistance is needed

Thank you for helping improve our security posture.

Security Operations Team
```
